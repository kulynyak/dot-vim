# dot-vim

My vim configuration

## Install
This vim config is available as [Ellipsis][Ellipsis] package.

#### Ellipsis
``` shell
    ellipsis install kulynyak/dot-vim
```

## Maintain
To keep everything up-to-date;

#### Ellipsis
```shell
    ellipsis pull vim
```
